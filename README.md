### Installing

#### Clone the project
```
git clone github.com/nicolas1bruno/listolistou-backend.git
```
and
```
cd listou-backend
```

#### Install dependencies
```
npm install
```

#### Config(connect to database)
create file '.env' in root folder and add code line
```
MONGO_URL=mongodb://localhost:27017/listou
```
<br/>
<br/>

### Start app

  npm run start
